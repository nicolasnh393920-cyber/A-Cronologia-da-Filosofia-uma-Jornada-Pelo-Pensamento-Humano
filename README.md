# A Cronologia da Filosofia: Uma Jornada Pelo Pensamento Humano

Este repositório contém um ebook abrangente e interativo que explora a cronologia da filosofia ocidental, desde suas origens até os pensadores contemporâneos. O projeto oferece diferentes formatos para acesso ao conteúdo, visando proporcionar uma experiência de leitura rica e acessível.

## Descrição do Projeto

"A Cronologia da Filosofia" é um recurso educacional detalhado, dividido em 25 capítulos, que guia o leitor através das principais escolas de pensamento, filósofos e conceitos que moldaram a história da filosofia. O objetivo é tornar o estudo da filosofia mais acessível e envolvente, combinando profundidade acadêmica com formatos de apresentação modernos.

## Funcionalidades

*   **Ebook Completo:** Conteúdo extenso com 25 capítulos sobre a história da filosofia.
*   **Formato PDF:** Uma versão tradicional e portátil do ebook para leitura offline.
*   **Versão HTML Interativa:** Uma experiência de leitura online aprimorada, com navegação intuitiva e design responsivo.
*   **Combo e Vendas:** Páginas dedicadas para a apresentação do combo de produtos e uma versão interativa para vendas.

## Como Acessar

Você pode acessar o conteúdo do ebook das seguintes formas:

*   **Visualizar Online:** Abra o arquivo `index.html` no seu navegador para a versão interativa.
*   **Download PDF:** O arquivo `A_Cronologia_da_Filosofia.pdf` está disponível para download.
*   **Páginas Específicas:**
    *   `entrega-html.html`: Página de entrega da versão HTML.
    *   `versao-interativa-vendas.html`: Versão interativa focada em vendas.
    *   `combo.html`: Página de apresentação do combo de produtos.
    *   `entrega-combo.html`: Página de entrega do combo.
    *   `pdf.html`: Página de acesso ao PDF.

## Tecnologias Utilizadas

O projeto é primariamente baseado em:

*   **HTML5:** Para a estrutura e conteúdo das páginas web.
*   **CSS3:** Para o estilo e design responsivo da interface.
*   **PDF:** Para a versão estática do ebook.
